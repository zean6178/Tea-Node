# hello-zean
A Simple Hello Zean Example Package

## Installation
Install via NPM:

```bash
npm install hello-zean

```

## Usage

#### javascript

```javascript

var myApp = require("hello-zean");

myApp.printMsg(); // print and return " Hello Zean! " 

```

#### TypeScript

```typescript

import * as myApp from 'hello-zean';

myApp.printMsg(); // print and return " Hello Zean! " 


```
